## #1 Requirement

- Pada kelas ini, kita akan coba membuat RESTful API untuk Contact Management, dimana RESTful API yang akan kita buat memiliki fitur sebagai berikut :
- User Management
- Contact Management
- Address Management

## #2 User Management Requirement

### User Data

- Username
- Password
- Name
- Token (Session Login)

### User API

- Register User
- Login User
- Update User
- Get User
- Logout User

## #3 Contact Management Requirement

### Contact Data

- First Name
- Last Name
- Email
- Phone

### Contact API

- Create Contact
- Update Contact
- Get Contact
- Search Contact
- Remove Contact

## #4 Address Management Requirement

### Contact Address Data

- Street
- City
- Province
- Country
- Postal Code

### Address API

- Create Address
- Update Address
- Get Address
- List Address
- Remove Address
